## امکانات

### بخش‌های محتوا
- **هیرو** — آتشگاه فراگیر با اثر تایپوگرافی و پارالاکس
- **داستان** — ۵ کارت قابل گسترش درباره عصر آتش
- **شخصیت‌ها** — ۲۹ شخصیت با جستجو، فیلتر و مودال جزئیات
- **اسطوره‌ها** — ۴ پرتره SVG فوق‌العاده (سولایر، سیگمایر، آرتوریاس، سیف)
- **مکان‌ها** — نقشه تعاملی پوست پوستی با ردیابی بازدید
- **باس‌ها** — ۲۶ باس با جستجو، فیلتر، مرتب‌سازی، امتیاز سختی و حالت مقایسه
- **عهدنامه‌ها** — ۹ جناح با راهنمای پیوستن و NPCهای مرتبط
- **آیتم‌های کلیدی** — ۱۲ آیتم نمادین با توضیحات
- **هدایای شروع** — ۵ گزینه هدیه شروع
- **خط زمانی** — ۷ رویداد تاریخی لوردان
- **نقل قول‌ها** — ۴۵ نقل قول چرخشی با اثر تایپ
- **محاسبه‌گر ساخت** — ۹ کلاس شروع با آمار قابل تنظیم
- **سلاح‌ها** — ۱۲ سلاح با جستجو و فیلتر
- **زره‌ها** — ۴۶ قطعه زره با توضیحات

### امکانات تعاملی
- جستجوی سراسری (Ctrl+K یا کلیک روی آیکون جستجو)
- تغییر حالت روشن/تاریک
- مودال مقایسه باس
- مودال جزئیات شخصیت
- پشتیبانی از ناوبری صفحه کلید
- ردیابی مناطق بازدید شده (localStorage)
- صفحه بارگذاری هنگام باز شدن

### افکت‌های بصری
- ذرات معلق محیطی (غبار، جرقه)
- افکت‌های پارالاکس پس‌زمینه
- افکت درخشش هنگام هاور روی کارت‌ها
- انیمیشن‌های نمایان شدن بخش‌ها
- لایه وینیت
- بافت نویز گرانول فیلم
- نوار پیشرفت اسکرول

## راهنمای استفاده

1. فایل `index.html` را در هر مرورگر مدرن باز کنید
2. از منوی هدر یا اسکرول برای ناوبری استفاده کنید
3. از آیکون جستجو (یا Ctrl+K) برای جستجو در تمام بخش‌ها استفاده کنید
4. حالت روشن/تاریک را با آیکون خورشید در هدر تغییر دهید
5. روی کارت‌های شخصیت کلیک کنید تا مودال جزئیات نمایش داده شود
6. از محاسبه‌گر ساخت برای برنامه‌ریزی ساخت شخصیت استفاده کنید
7. سلاح‌ها و زره‌ها را بر اساس نوع جستجو و فیلتر کنید

## ایسترهای تخم‌مرغی

### میانبرهای صفحه کلید
| میانبر | اثر |
|--------|------|
| `↑↑↓↓←→←→BA` | کد کونامی — فلش چرخش رنگ |
| تایپ `DARK` | تاریک شدن صفحه با کاهش اشباع |
| تایپ `SOUL` | درخشش آبی روح |
| تایپ `ILLUSORY` | پیام "دیوار دروغینی در پیش است..." |
| تایپ `BONFIRE` | رنگ نارنجی گرم |
| تایپ `HOLLOW` | هشدار "تنه نشو..." |

### ایسترهای کلیکی
| عمل | اثر |
|-----|------|
| ۵ بار کلیک روی آتشگاه | انیمیشن شعله آتشگاه |
| اسکرول بیش از حد (۵۰۰+) | پوشش "YOU DIED" |
| بازدید از ۲۰+ مکان | نقل قول مخفی در فوتر |
| ۳ بار کلیک روی کارت سولایر | "PRAISE THE SUN!" |
| ۳ بار کلیک روی کارت سیگمایر | "Hmm... What's this?" |
| ۳ بار کلیک روی کارت آرتوریاس | اثر غرش اbyssal |
| ۳ بار کلیک روی کارت سیف | پیام وفاداری |

## تکنولوژی‌های استفاده شده

- **HTML5** — نشانه‌گذاری معنایی
- **CSS3** — ویژگی‌های سفارشی، گرادیانت، انیمیشن، backdrop-filter
- **JavaScript خالص** — بدون فریمورک یا کتابخانه
- **SVG** — تصاویر برداری برای شخصیت‌ها و آیکون‌ها
- **localStorage** — ردیابی پیشرفت و تنظیمات
- **Service Worker** — پشتیبانی آفلاین
